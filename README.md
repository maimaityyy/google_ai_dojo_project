# EmojiStamp

EmojiStamp は、写真に絵文字スタンプを貼ったり、手書きでデコレーションしたりできる Android アプリです。

## 主な機能

- **写真の選択・撮影**: Photo Picker または標準カメラアプリを使用して、加工する写真を取り込めます。
- **絵文字スタンプ**: 6 種類の絵文字を好きな場所にタップで配置できます。配置後のドラッグ移動や長押しでの削除も可能です。
- **手書き編集（ペン機能）**: 写真の上に自由に線を引くことができます。
- **消しゴム機能**: 描いた線をなぞって消すことができます。

## 手書き編集機能の工夫点

今回の実装では、スムーズで直感的な操作感を実現するために以下の点にこだわっています。

### 1. 高精度な描画範囲の制限（Dynamic Clamping）
単なる固定サイズの枠ではなく、読み込まれた写真の**実際のアスペクト比に合わせて描画領域を動的に計算**しています。
- 写真の横や上下にある空白部分をタップしても反応せず、境界線でピタッと線が止まるように制御しています。
- `ContentScale.Fit` のロジックに基づいた正確なピクセル計算を行っています。

### 2. 直感的な消しゴムロジック
ドット単位で消すのではなく、**「描いた線をなぞるとそのストロークが消える」**方式を採用しています。
- タッチ座標と描画パスの近傍判定を行い、意図した線をストレスなく一気に消せるように設計しました。

### 3. 滑らかな描き心地の実現
`Canvas` と `Path` を組み合わせ、以下の設定を施すことで、指の動きに追従する滑らかな線描を実現しています。
- `StrokeCap.Round` および `StrokeJoin.Round` を指定し、線の継ぎ目や終端を丸く処理しています。

### 4. モードに応じたジェスチャの共存
`detectTapGestures` と `detectDragGestures` を組み合わせ、モード（スタンプ/ペン）に応じて以下の操作をインテリジェントに切り替えています。
- **スタンプモード**: タップ（配置）、長押し（削除）、ドラッグ（移動）
- **ペン・消しゴムモード**: ドラッグ（描画・消去）

### 5. 状態の堅牢性
`rememberSaveable` や `mutableStateListOf` を活用し、画面の回転やテーマの切り替えが発生しても、作成中のスタンプや手書きの線が消えることなく保持されるように実装しています。

## 技術スタック
- **UI**: Jetpack Compose (Material 3)
- **Image Loading**: Coil 3
- **Gesture**: Pointer Input (detectTapGestures, detectDragGestures)
- **Architecture**: Stateless Composables with State Hoisting
